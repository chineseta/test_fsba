Flight Status By Airport (with flightstats.com)

Для работы нужны (установлено у меня): Linux (Debian 6), Nginx (0.7.67), Node.js (0.8.9), Redis (2.4.17)
Установка пакетов: npm install
Запуск тестов: make или make test
Запуск node сервера: make run (на VPSке используется runit)
Папка conf/: конфиги для VPSки
Папка spec/: тесты на jasmine

* записи можно сортировать, при наведении на codeshare показывается подсказка
